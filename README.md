# Cahaya Cargo Express - C2MS

Sistem manajemen logistik berbasis Next.js dan Firebase.

## Instalasi

```bash
npm install
npm run dev
```

Buka http://localhost:3000

## Fitur

- ✅ Database Client (CRUD, Search, PKP Status)
- ⏳ Resi & Transaksi
- ⏳ Manifest & Tracking
- ⏳ Dashboard BI

## Tech Stack

- Next.js 15 + TypeScript
- Firebase (Firestore)
- Tailwind CSS
